# student-management-frontend
 
## Setup

 1. git clone this repository
 ```
 git clone https://github.com/dev-chester/student-management-frontend.git student-management-frontend-ca
 ```
 
 2. install dependencies
 ```
 npm install
 ```
 
## Running

 - Run locally
 ```
 npm run dev
 ```
 
 - Run via Vercel
 https://student-management-frontend-cbfi7bdl4-chesterarellano.vercel.app/
